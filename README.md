# 纯前端版校园学习资源共享站

## 项目简介

本项目是一个**纯前端、无需后端服务器**的校园学习资源共享平台，旨在解决师生在学习资料流转过程中遇到的文件过期、分类混乱、检索困难等常见问题。采用 HTML 、CSS 和 JavaScript 技术栈实现，所有数据均存储在浏览器本地，注重隐私与安全，开箱即用。

## 核心功能

1.  **资料分类上传**
    *   支持多种格式：PDF 、Word 、PPT 、图片、音频、TXT 。
    *   三级分类体系：学段 -> 学科 -> 资料类型。
    *   支持为资料添加自定义标签，便于精细化检索。

2.  **智能检索与预览**
    *   支持关键词检索（匹配文件名、标签）。
    *   提供多条件组合筛选（学段、学科、类型、标签）。
    *   支持在线预览 PDF 、图片 、TXT ，以及播放音频文件。

3.  **灵活导出与备份**
    *   可单份导出原始文件。
    *   支持批量选择资料，打包为 ZIP 压缩文件下载。
    *   提供全量数据备份与恢复功能（导出/导入 JSON ），便于资料迁移。

4.  **贡献统计与激励**
    *   统计个人上传与下载总量。
    *   可视化展示“资源贡献榜”，激励分享。
    *   自动标记下载量高的资料为“优质资源”。

## 技术栈

*   **前端三件套**: HTML5 、CSS3 、JavaScript
*   **UI 框架**: Bootstrap 5
*   **核心存储**: 浏览器 LocalStorage
*   **关键开源库**:
    *   `PDF.js`: 实现 PDF 文件在线预览。
    *   `JSZip`: 实现前端文件批量打包为 ZIP 。
    *   `Chart.js`: 绘制贡献统计图表。

## 快速开始

1.  **获取项目**：克隆或下载本项目到本地。
2.  **运行项目**：直接在浏览器中打开 `index.html` 文件，无需服务器环境。
3.  **开始使用**：界面加载后，即可体验上传、检索、预览、导出全部功能。

## 注意事项

*   **存储限制**：由于使用浏览器 LocalStorage ，单个源站点的存储空间有限（通常为5-10MB），适合分享数量适中、体积较小的学习资料。
*   **数据持久性**：清理浏览器缓存可能导致本地数据丢失，请定期使用“数据备份”功能导出重要资料库。
*   **兼容性**：建议使用 Chrome 、Edge 、Firefox 等现代浏览器以获得最佳体验。
